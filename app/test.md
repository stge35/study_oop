# PySide6 사용방법

### QApplication 
- 앱당 1개만 생성
- 앱의 전체 라이프 사이클과 이벤트를 총괄 관리
- 극장 감독(배우와 무대가 움직이도록 관리)

### QMainWindow
- 메뉴바, 상태바, 툴바, 메인 영역을 내장 하고 있는 표준 메인 창틀

### QWidget
- 가장 기본이 되는 빈 도화지
- 레이아웃을 담는 컨테이너 혹은 위젯을 만들 때 베이스로 씀

### QVBoxLayout
- 위젯들을 위에서 아래로 차례대로 자동 배치해주는 레이아웃 관리자

### QLabel
- 화면에 텍스트나 이미지를 보여줄 때 사용하는 읽기 전용 위젯

### QPushButton
- 버튼을 만드는 위젯
- 클릭 시 시그널을 발생
